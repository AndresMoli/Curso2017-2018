Analysis of Data Set

Agenda Actividades Eventos

Column Name Type Value Range Description
ID-EVENTO Integer [0-n] Identificador del evento.
TITULO String [0-9 a-Z] Título del evento.
PRECIO String [0-9 a-Z] Precio del evento.
GRATUITO Boolean 0 ó 1 Indica si el evento es gratuito o no.
LARGA-DURACION Boolean 0 ó 1 Indica si la duración del evento es larga o no.
DIAS-SEMANA String [0-9 a-Z] Indica los días de la semana en los que se realiza el evento.
DIAS-EXCLUIDOS String [0-9 a-Z] Indica los días de la semana en los que no se realiza el evento.
FECHA Date (dd/mm/yyyy) [01/01/2000 - 31/12/2017] Fecha del evento.
FECHA-FIN Date (dd/mm/yyyy) [01/01/2000 - 31/12/2017] Fecha de la finalización del evento.
HORA Hora (HH:mm) [0:00 - 23:59] Hora en la que se realiza el evento.
DESCRIPCION String [0-9 a-Z] Descripción del evento.
CONTENT-URL String [0-9 a-Z] Enlace del evento.
TITULO-ACTIVIDAD String [0-9 a-Z] Título de la actividad que se realiza.
URL-ACTIVIDAD String [0-9 a-Z] Enlace de la actividad que se realiza.
URL-INSTALACION String [0-9 a-Z] Enlace de las instalaciones donde se realiza el evento.
NOMBRE-INSTALACION String [0-9 a-Z] Indica el nombre de las instalaciones donde se realiza el evento.
COORDENADA-X Integer [0-n] Indica la coordenada X donde se realiza el evento.
COORDENADA-Y Integer [0-n] Indica la coordenada Y donde se realiza el evento.
LATITUD Integer [0-n] Indica la latitud donde se realiza el evento.
DISTRITO String [0-9 a-Z] Distrito donde se realiza el evento.
TIPO String [0-9 a-Z] Tipo de evento que se realiza el evento.
AUDIENCIA String [0-9 a-Z] Audiencia a la que va dirigida el evento.

Aparcamientos Públicos

Column Name Type Value Range Description
PK Integer [0-n] Identifica el aparcamiento público.
NOMBRE String [0-9 a-Z] Nombre del aparcamiento.
DESCRIPCION-ENTIDAD String [0-9 a-Z] Breve descripción del aparcamiento.
HORARIO String [0-9 a-Z] Horario del aparcamiento.
EQUIPAMIENTO String [0-9 a-Z] Indica el equipamiento del aparcamiento público.
TRANSPORTE String [0-9 a-Z] Transporte hasta el aparcamiento.
DESCRIPCION String [0-9 a-Z] Datos identificativos del aparcamiento.
ACCESIBILIDAD Boolean 0 ó 1 Indica si el aparcamiento tiene accesos para gente discapacitada.
CONTENT-URL String [0-9 a-Z] Enlace del aparcamiento.
NOMBRE-VIA String [0-9 a-Z] Nombre de la vía en la que se encuentra el aparcamiento.
CLASE-VIAL String [0-9 a-Z] Tipo de vía en la que se encuentra el aparcamiento.
TIPO-NUM String V ó NUM
NUM Integer [0-n] Datos identificativos del aparcamiento.
PLANTA String [0-9 a-Z]
PUERTA String [0-9 a-Z]
ESCALERAS String [0-9 a-Z]
ORIENTACION String [0-9 a-Z] Orientación del aparcamiento.
LOCALIDAD String [0-9 a-Z] Localidad en la que se sitúa el aparcamiento.
PROVINCIA String [0-9 a-Z] Provincia en la que se sitúa el aparcamiento.
CÓDIGO-POSTAL String [0-9 a-Z] Indica el código postal del aparcamiento.
BARRIO String [0-9 a-Z] Barrio en la que se sitúa el aparcamiento.
DISTRITO String [0-9 a-Z] Distrito en la que se sitúa el aparcamiento.
COORDENADA-X String [0-9 a-Z] Coordenada X donde se sitúa el aparcamiento.
COORDENADA-Y String [0-9 a-Z] Coordenada Y donde se sitúa el aparcamiento.
LATITUD Integer [0-n] Latitud donde se sitúa el aparcamiento.
LONGITUD Integer [0-n] Longitud donde se sitúa el aparcamiento.
TELEFONO Integer [0-n] Télefono del aparcamiento.
FAX String [0-9 a-Z] Fax del aparcamiento.
EMAIL String [0-n] Email del aparcamiento.
TIPO String [0-n] Tipo del aparcamiento.
